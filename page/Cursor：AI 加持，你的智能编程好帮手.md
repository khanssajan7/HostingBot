## 快速上手 Cursor

### 安装与配置
1. **下载安装**  
   - 访问 Cursor 官网下载安装包  
   - 运行安装程序完成安装  
2. **迁移 VS Code 配置**

3. **配置 AI 助手**  
   - 选择喜欢的 AI 模型  
   - 配置 API 密钥（如需要）  

### 常用快捷键
- `Cmd/Ctrl + I`：开启 AI 助手，进行自然语言对话  
- `Cmd/Ctrl + K`：快速执行 AI 提示（代码生成、重构等）  
- `Cmd/Ctrl + L`：让 AI 为你解释代码  
- `Cmd/Ctrl + Enter`：接受 AI 的建议  
- `Esc`：拒绝 AI 的提议
